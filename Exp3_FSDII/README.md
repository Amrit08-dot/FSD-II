## Learning Outcomes

By completing this project, the following learning outcomes were achieved:

- Developed a strong understanding of **React components**, including their role in building modular, reusable, and maintainable user interfaces.

- Learned how to properly **configure and manage `App.jsx` in React**, including setting up the application structure, importing components, managing component hierarchy, and controlling the main application flow.

- Gained practical experience in **applying and integrating components** within a React application to build a cohesive and functional user interface.

- Understood the principles of **Single Page Applications (SPA)** and how React efficiently updates the UI without full page reloads.

- Implemented **client-side routing** using React Router to enable smooth navigation between different views or pages within the SPA.

- Learned how to define and manage **routes, links, and navigation components**, improving user experience through seamless page transitions.

- Enhanced knowledge of **state management and component interaction**, ensuring consistent data flow across routed components.
